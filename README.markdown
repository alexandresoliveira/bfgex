regex based generation
=========================================

 use like a maven dependency on your project

 	  <dependency>
	     <groupId>br.com.six2six</groupId>
		 <artifactId>bfgex</artifactId>
		 <version>1.0</version>
	   </dependency>

usage
=======================================

RegexGen.of("(\\w{4})(\\d{3})"); //return zori873
